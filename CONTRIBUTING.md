🚀 Guia de Contribuição
Olá! Que bom que você quer contribuir com este projeto. Siga os passos abaixo para criar sua versão e enviar suas melhorias.

1. Como começar
Faça um Fork deste repositório (botão no canto superior direito).
Clone o seu fork para sua máquina: git clone https://github.com

2. Desenvolvendo sua versão
Antes de alterar o código, crie uma nova branch (ramificação) para organizar seu trabalho: git checkout -b minha-contribuicao

Dica: Use nomes curtos que descrevam o que você está fazendo (ex: ajuste-layout, correcao-calculo).

3. Salvando as alterações
Após fazer as mudanças no código:
Adicione os arquivos: git add .
Crie um commit com uma mensagem clara: git commit -m "Explique aqui o que você mudou ou adicionou"

Envie para o seu GitHub: git push origin minha-contribuicao


4. Enviando (Pull Request)
Quando estiver pronto para que eu veja seu trabalho:
Vá até a página do seu fork no GitHub.
Clique no botão "Compare & pull request".
Descreva brevemente o que você implementou e clique em "Create pull request".

💡 Dicas importantes:
Mantenha seu código limpo: Comente partes complexas se necessário.
Sincronização: Se o projeto original mudar, não esqueça de atualizar seu fork.
Dúvidas? Abra uma Issue no repositório principal!
